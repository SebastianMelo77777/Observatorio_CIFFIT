# Observatorio Fiscal Javeriana - CUIPO Ejecución de Gastos

Automatización en Python para descargar, verificar y procesar la ejecución de gasto CUIPO desde datos.gov.co.

## Estructura del proyecto
- `src/` → código modular
- `config/config.yaml` → configuración
- `main.py` → script principal
- `data/processed/` → archivos limpios en Parquet
